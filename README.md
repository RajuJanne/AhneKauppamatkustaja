# AhneKauppamatkustaja
*	Kauppamatkustajan ongelmaksi kutsutaan tilannetta, jossa kauppamatkustajan on matkustettava n:n kaupungin kautta siten, että matka on lyhin mahdollinen.
* Toteuta luokka, jonka avulla ongelma voidaan mallintaa. Ratkaise ongelma käyttäen ahnetta algoritmia (greedy algorithm).
